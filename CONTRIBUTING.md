# Contributing

This is a starter template -1. **Fork the Repository**: Click the "Fork" button at the top right of the repository page. 2. **Clone your Fork**:
`bash
    git clone https://github.com/Nihal-Pandey-2302/lazorkit-solana-starter.git
    cd lazorkit-solana-starter
    ` 3. **Create a Branch**:
`bash
    git checkout -b feature/amazing-feature
    ` 4. Install deps: `npm install` 5. Run dev server: `npm run dev`

## Pull Request Guidelines

- Follow existing code style (Prettier config included)
- Add tests for new features
- Update README if adding new functionality
- Keep commits atomic and well-described

## Reporting Issues

Found a bug? [Open an issue](https://github.com/YOUR_USERNAME/solana_passkey_integration/issues)

Include:

- Steps to reproduce
- Expected vs actual behavior
- Browser/OS version
